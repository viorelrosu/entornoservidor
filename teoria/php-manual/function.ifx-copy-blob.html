ifx_copy_blob

(PHP 4, PHP 5 < 5.2.1)

ifx_copy_blob — Duplicates the given blob object

Description

ifx_copy_blob ( int $bid ) : int

Duplicates the given blob object.

Parameters

bid

A BLOB identifier.

Return Values

Returns the new blob object-id, or FALSE on errors.
