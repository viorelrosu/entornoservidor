ifxus_close_slob

(PHP 4, PHP 5 < 5.2.1)

ifxus_close_slob — Deletes the slob object

Description

ifxus_close_slob ( int $bid ) : bool

Deletes the slob object on the given slob object-id bid.

Parameters

bid

An existing slob id.

Return Values

Returns TRUE on success or FALSE on failure.
